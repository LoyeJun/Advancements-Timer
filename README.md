### 想要拥有Minecraft主题的Windows体验吗？
那就试试这款能显示时间的 **小工具** 吧！

---
```
尽管目前还存在着一些Python API的诡异问题尚未解决
详细版本和更新内容可查看Releases
```
---

### 功能：
1. **MC主题时间显示**：在窗口上展示Minecraft风格的当前时间
2. **设定闹钟**：可以设定一个闹钟

### 待修复Bug：
1. **启动失败**：有时启动程序可能会失败
2. **窗口漂移**：窗口有时会出现位置偏移的问题
3. **字体模糊**：字体显示可能不够清晰
4. **闹钟格式限制**：闹钟时间仅支持HH:MM:SS格式
